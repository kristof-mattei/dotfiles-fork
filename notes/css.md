# CSS

## CSS Grid

### Parent

- `display: grid;`
- column/row specification example: `grid-template-columns: 1fr 50px 20%;`

### Children

- Placement: `grid-column: 1 / 3;`


## Flex

### Parent

- `display: flex`
- Horizontal Spacing
    - `justify-content`

### Children

- `flex: grow-factor shrink-factor default-size`
