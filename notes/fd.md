# fd

Find all results: `fd -H -I`. -H for hidden, -I for ignored. Smart Case
by default.

```
{} path
{/} basename (aka filename)
{//} parent directory
{.} path without file extension
{/.} basename without file extension
```

```
-d, --max-depth <depth>
-t, --type <type>  f, d, l, x, e (empty)
-x, --exec <cmd>
```
