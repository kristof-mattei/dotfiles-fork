## Installing

Ubuntu (among others) [link](https://www.atechtown.com/install-fonts-in-ubuntu/):

Copy to folder:
 - Installing for single user, `~/.fonts`
 - All Users: `/usr/local/share/fonts`

Refresh font cache:

```
sudo fc-cache
```
